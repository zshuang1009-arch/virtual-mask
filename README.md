# Virtual Mask - 虚拟面具

一个探索数字身份与外貌焦虑的互动网站

## 一键部署

### 前置要求
- Node.js (建议版本 14 或更高版本

### 快速启动

#### Windows 用户:
直接双击 `启动网站.bat` 文件

#### 或使用命令行:
```bash
npm start
```

### 手动启动:
```bash
node server.js
```

### 访问地址
启动后在浏览器中打开: **http://localhost:3000

## 项目结构

```
Virtual Mask/
├── index.html          # 首页
├── folder1.html       # 文件夹1页面
├── folder2.html       # 文件夹2页面 (美颜编辑器)
├── folder3.html       # 文件夹3页面 (手势面部识别)
├── style.css         # 样式文件
├── script.js         # 脚本文件
├── obstacle.js       # 障碍系统
├── server.js         # 本地服务器
├── 启动网站.bat       # 一键启动脚本
├── 图标/             # 图标资源
├── 障碍/             # 障碍图片
├── 面部拼图/         # 面部拼图资源
└── *.mp4            # 视频资源
```

## 功能说明

1. **首页**
   - 三个文件夹图标
   - 终端打字效果
   - 信息窗口

2. **Folder 1**
   - 视频展示
   - 理论介绍
   - 面部拼图游戏
   - 照片查看器

3. **Folder 2**
   - 实时美颜编辑器
   - 美白、磨皮、轮廓、色温调整

4. **Folder 3**
   - 面部识别
   - 像素打乱
   - 手势交互

5. **障碍系统**
   - 点击屏幕生成障碍
   - 障碍在页面间持久化
   - 点击障碍可消除
   - 50个障碍后显示"探索失败"

## 技术栈
- HTML5
- CSS3
- JavaScript (ES6+)
- Node.js (服务器)
- TensorFlow.js (面部识别)
